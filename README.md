## To get started

1. Clone the repo
```bash
git clone https://github.com/darbandi/block-chain.git
```
2. Install dependencies using Yarn
```bash
cd block-chain
yarn
```

3. Start the development server
```bash
yarn dev
```

4. Start the Testing
```bash
yarn test
```
